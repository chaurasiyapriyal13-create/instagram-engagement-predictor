##Instagram enagament predictor (ML mini project ) for collage 

## Project Overview
This project predicts Instagram post engagement (likes) using Machine Learning.

The model analyzes multiple post attributes such as reach, impressions, caption length, hashtags, and media type to understand factors affecting engagement.

## Dataset
Instagram Analytics Dataset containing ~30,000 posts with features including:

- Likes
- Comments
- Shares
- Saves
- Reach
- Impressions
- Caption Length
- Hashtags Count
- Media Type
- Content Category

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Machine Learning Model
Random Forest Regressor was used to predict Instagram likes based on post features.

## Project Workflow
1. Data Cleaning
2. Feature Engineering
3. Exploratory Data Analysis
4. Model Training
5. Model Evaluation

## Key Insights
- Reach and impressions significantly influence engagement
- Posts with higher shares and saves tend to receive more likes
- Hashtags and caption length moderately impact performance
